# 🧪 ES6 e Testes Unitários

O **ES6 e Testes Unitários** é um projeto focado no desenvolvimento de algoritmos em JavaScript moderno e na aplicação prática de **Testes Unitários** e **Desenvolvimento Orientado a Testes (TDD)** utilizando o framework Jest. 

Neste projeto, implementei diversas funções lógicas na pasta `src` e construí suites de testes robustas na pasta `tests` para assegurar o funcionamento pleno de cada funcionalidade, eliminando falsos positivos e garantindo a qualidade do código.

---

## 🚀 O que foi desenvolvido

O projeto engloba soluções lógicas completas e a criação de testes de cobertura. As principais atividades desenvolvidas foram:

*   **Implementação de Algoritmos:** Desenvolvimento de funções como `average` (cálculo de médias com arredondamentos e tratamentos de erros), `vqv` (geração de apresentações textuais usando *template literals*), `createStudent` (fábrica de objetos com funções de feedback), `calculator` e `arrayGenerator` (operações matemáticas e conversores de objetos em arrays), além de refatorações de loops com `myCounter`.
*   **Criação de Testes Unitários com Jest:** Escrita de casos de testes rigorosos para funções pré-existentes como `numbers` (validação de tipos primitivos), `circle` (cálculo de propriedades geométricas), `productDetails` (análise de metadados de arrays de objetos) e `getCharacter` (busca de personagens com comportamento *case-insensitive*).
*   **Fluxo Completo de TDD (Test Driven Development):** Desenvolvimento guiado por testes para a função `createMenu` (`src/restaurant.js`). O processo envolveu a criação alternada de testes e código para gerenciar o menu de um restaurante, controlar o histórico de consumo via métodos (`order`), tratar itens indisponíveis e calcular a conta final acrescida de 10% de taxa de serviço (`pay`).

---

## 🛠️ Tecnologias Utilizadas

*   **JavaScript (ES6):** Uso de *Template Literals*, Arrow Functions, manipulação avançada de objetos e tratamento de exceções.
*   **Node.js:** Ambiente de execução local da aplicação.
*   **Jest:** Framework de testes utilizado para criar asserções, mockar comportamentos e rodar a cobertura automatizada.
*   **ESLint:** Linter para a análise estática do código, assegurando a padronização e as boas práticas de desenvolvimento.
